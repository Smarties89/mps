# rstr
rstr creates a random string of numbers and upper/lowercase characters.

```
>>> import mps
>>> mps.rstr()
'N5KZRC06'
>>> mps.rstr(nsize=100)
'5QRDHYESVHMNL68E0IPYEIP3TXBUU86V26PSIEZ4AWE1FAX3G2WCSJ38NAIHB9DM3ZCCI102AI6TAOC967XM3UW6ZH5X9S8OQR23'
>>> 
```
